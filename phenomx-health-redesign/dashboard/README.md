# PhenomX Health Dashboard

An interactive health tracking dashboard built with React.

## Features
- Menstrual cycle tracking
- Weight & BMI monitoring
- Symptom tracking with visualizations
- Food intake logging
- Daily motivation & goals
- Sleep program guidance
- Nutritional advice

## Tech Stack
- React 18
- Recharts for data visualization
- Lucide React for icons
- Tailwind CSS for styling
